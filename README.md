Dotfiles and other system stuff

# Installing
* From the repo root, run `install.sh`
* Add the following to `.profile` on Mac or `.bashrc` on Linux:
  `source ~/.bash_common`
* Source `.profile` or `.bashrc`, or open up another terminal window
